# VuePress 社区

社区维护的 VuePress 生态系统。

# 插件

名称 版本 介绍
clean-urls 💧 在 VuePress 中使用简洁链接
container 📦 在 VuePress 中使用 markdown 容器
copyright 📋 处理 VuePress 中的复制操作
dehydrate 🔥 修改 VuePress 生成的 HTML 文件
git-log 💾 在 VuePress 中集成 git 日志
mathjax 📃 在 VuePress 中使用 TeX 语法
medium-zoom 🔍 使 VuePress 中的图片支持缩放
migrate 🐾 从其他网站迁移到 VuePress
named-chunks 📛 在 VuePress 中使用命名模块
nprogress 🏃 在 VuePress 中显示页面进度条
redirect 🚥 处理 VuePress 中的重定向问题
serve 🔑 在本地构建静态 VuePress 服务器
smooth-scroll 🎢 在 VuePress 中使用平滑滚动
table-of-contents 🔖 为 VuePress 提供目录组件
typescript 🌴 在 VuePress 中支持 TypeScript
zooming 🔍 使 VuePress 中的图片支持缩放
